# Comprehensive Design Guidelines

## Principles
- Clarity over complexity
- Evidence-first communication
- Consistent interactions

## Color guidance
- Primary text: `slate-900`
- Secondary text: `slate-600`
- Surface: `white` / `slate-50`
- Positive: `emerald-*`
- Warning: `amber-*`
- Error: `red-*`

## Component behavior
- Table sorting for comparisons
- Stable chart/table toggles
- No duplicate fullscreen controls
- Clear empty states

## Accessibility
- Adequate contrast
- Keyboard usability
- Avoid color-only meaning
